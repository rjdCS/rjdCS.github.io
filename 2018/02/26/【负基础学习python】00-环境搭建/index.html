1.安装Anaconda

  • 安装之前

    Q：为什么要安装Anaconda？
    A：回答这个问题我们先回答下面这个问题

Anaconda是什么？
Anaconda翻译过来的意思是“蟒蛇”，python也是“蟒蛇”的意思。

简单点来说，Anaconda就是python的环境管理，也就是python环境（当然没有这个简单啦，目前来说可以这么理解）
回到上面的那个问题：

Q：为什么要安装Anaconda？
A：可以方便的运行和管理你自己的python

  • 事实上Anaconda 和 Jupyter notebook（Anaconda内置应用）已成为数据分析的标准环境。
    不安装行不行？
    当然可以！前提是你会。

正式安装

需要：

  • 一台连上网的电脑（windows系统）
    <<<楼上不是废话嘛啊喂！>>>

步骤：

  1. 打开百度，输入 清华镜像 （注意网址，红色的），点击进入。

2.选择Anaconda（红色方框），点击进入。

3.选择archive，点击进入。

4.选择Anaconda3-4.2.0-Windows-x86_64.exe，点击即可下载。

5.下载完成后，双击下载好的Anaconda文件。

6.进入安装程序。

7.一直点击Next，直到下图所示页面，一定要确保红色方框内打钩了！点击Install进行安装。

8.安装完毕后，将“Learn more about Aanaconda Cloud”前的对号去掉点击Finish完成安装。

9.回到桌面，按住win+R键打开运行，输入cmd（命令控制行，知道就行，不用管，以后经常会用），点击确定打开。

2.使用jupyter notebook教程

jupyter notebook是一个可以运行python程序的网页程序，非常方便。是数据分析的首选。

  • 打开jupyter notebook
    1.打开cmd（win+R，输入cmd，回车即可）
    2.输入jupyter notebook，如下图则表示打开成功，会自动打开网页。

  • 使用jupyter notebook
    1.单击右上角的new。

2.点击Folder，会生成一个Untitled Folder

3.选择文件夹（Untitled Folder）前的小框，打钩，点击左上角的Rename，另取名为python_learn。点击OK完成。

4.打开python_learn文件夹（单击打开）。

5.右上角点击new，然后点击Python[default]，新建一个python文件。

6.单击最上面的Untitle修改文件名。修改名为hello_world，点击OK。

7.jupyter简单用法搞定！
